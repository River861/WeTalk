## 简介

这是一个通过socket编程简单实现的聊天室，主要功能有快速推出、历史记录、撤回消息、发送表情图片、群主踢人特权与轮转制、文件传输。传输前进行了密钥交换，消息传输使用了AES加密传输。界面通过pyqt5+html渲染。



## 运行环境与指令

运行环境：Ubuntu 18.04、Python 3.6、PyQt5

运行指令：

 * 服务端启动：python server.py

 * 客户端启动：python WeTalk.py
